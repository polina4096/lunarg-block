# lunarg-block

Adds a block craftable from 9 vulkan triangles which scares minecraft players.

https://x.com/NearestCommit/status/2024946823819878695

### Build Instructions

1. Clone the git repository
2. Navigate into the cloned directory
3. Run `./gradlew build`

The built jar file will be in `build/libs/`
